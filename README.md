# infopoisk_hw1
1. В качестве датасета, я взяла свою собственную базу данных, которую делала около 2 лет назад. В ней собраны песни одного исполнителя, данные были получены путём краулинга сайта Genius. В базе данных содержится информация по песням (альбомы, год выпуска), тексты песен, как полностью так и разбитые построчно, а также отдельные таблицы по частям речи. Для задания я использую только часть данных, а конкретно тексты песен, разбитые на строки. Чтобы сократить размер датасета, я беру только строки, в которых есть минимум 6 слов, а также убираю повторяющиеся строчки. Помимо этого исключила строки, в которых текст был на испанском и английском языках. В итоге получилось 2102 записей
